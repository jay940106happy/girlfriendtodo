# Love Todo Memory

A romantic Vue 3 + Vite + Supabase app for shared couple todos and memory journaling.

## Setup

1. Copy `.env.example` to `.env.local`
2. Fill in your Supabase URL and publishable key
3. Run the SQL in `supabase/setup.sql`
4. Install dependencies with `npm install`
5. Start with `npm run dev`

## Deploy

Deploy to Vercel as a Vite project.

Environment variables:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
